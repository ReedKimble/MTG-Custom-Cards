# MTG-Custom-Cards
Magic the Gathering Custom Card Sets

## View the Cards

Visit the [Card Gallery](https://reedkimble.github.io/MTG-Custom-Cards/) to browse all custom card sets in a beautiful, easy-to-navigate interface.

## Card Sets

This repository contains the following custom Magic: The Gathering card sets:

- **Eternia** - 42 cards inspired by He-Man and the Masters of the Universe
- **Urza and the Ozolith** - 61 cards featuring Arcbound and Ozobound mechanics

## About

These are custom Magic: The Gathering cards created for fun and creativity. All card images are located in the `Mini-Sets/` directory.
